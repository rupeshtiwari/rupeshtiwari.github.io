<p>In order to install required angular material, angular team has provided six steps to follow in angular material home page. All of them are listed when we click on “Get Started” on home page.</p>
<p><img class="alignnone size-full wp-image-2785" src="{{ site.baseurl }}/assets/2019/11/AMC1.png" alt="" width="698" height="389" /></p>
<p>However performing all these six steps manually is prone to errors, so angular team has provided single line command to do all the six steps in one go. It will change all required files in our project accordingly and make our life easier.</p>
<p><img class="alignnone size-full wp-image-2784" src="{{ site.baseurl }}/assets/2019/11/AMC2.png" alt="" width="698" height="392" /></p>
<p>Complete installation process is explained in our demo session <a href="https://www.youtube.com/watch?v=ek63pA4rqvA&amp;list=PLZed_adPqIJrl9pwlERGhU-RCNOtKqvyD&amp;index=7" target="_blank" rel="noopener noreferrer"><strong>Installing Angular Material Using Angular CLI | Tutorial | 0009-07 </strong></a>in detail. It shows how with single command we will integrate angular material. After running <strong>ng add @angular/material</strong> command it will start installation process and ask you to choose theme and say yes/no for gesture recognition/animation etc.</p>
<p><img class="alignnone size-full wp-image-2783" src="{{ site.baseurl }}/assets/2019/11/AMC3.png" alt="" width="698" height="390" /></p>
<p>We can revisit our code and verify what are all changes have been done after angular material installation is done.</p>
<p><img class="alignnone size-full wp-image-2782" src="{{ site.baseurl }}/assets/2019/11/AMC4.png" alt="" width="699" height="394" /></p>
<p>Also if you check index page, you will see fonts/icons have been added.</p>
<p><img class="alignnone size-full wp-image-2781" src="{{ site.baseurl }}/assets/2019/11/AMC5.png" alt="" width="696" height="391" /></p>
<p>Like angular material, angular has separate <strong>Material Icons</strong> site where you can focus more on Icon/fonts etc accessibility stuff. These all are open source and we can access them easily from this site.</p>
<p><img class="alignnone size-full wp-image-2780" src="{{ site.baseurl }}/assets/2019/11/AMC6.png" alt="" width="698" height="396" /></p>
<p>One thing that was missing in this automated angular material installation process is it does not install any component modules. This is because; it doesn’t know what your requirement is. This we need to do manually, we will see how to do it our next demo session <a href="https://www.youtube.com/watch?v=NH4U0_PCTAk&amp;list=PLZed_adPqIJrl9pwlERGhU-RCNOtKqvyD&amp;index=8" target="_blank" rel="noopener noreferrer"><strong>Angular Material Components | Demo | 0009-08</strong></a></p>
