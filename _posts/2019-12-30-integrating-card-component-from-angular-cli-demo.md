<p>This demo session  <a href="https://www.youtube.com/watch?v=BETUMZgepeo&amp;list=PLZed_adPqIJrl9pwlERGhU-RCNOtKqvyD&amp;index=13&amp;t=0s" target="_blank" rel="noopener noreferrer"><strong>Applying Card Component | Angular Material | Demo | 0009-12 </strong></a>talks about integrating card components from angular material so that we can put them in our app to showcase our products list. In order to get source code for required cards, we will open angular material site and go to card section inside layout.</p>
<p><img class="alignnone size-full wp-image-2855" src="{{ site.baseurl }}/assets/2019/12/CC1.png" alt="" width="855" height="484" /></p>
<p>Now we will place this copied source code in our <strong>products.component.html </strong>and make some changes to make it suitable for our project. Changes we need to do are:</p>
<ul>
<li>First we need to pull angular material module into our <strong>module.ts</strong> as an import, so that our angular materials are recognized.</li>
</ul>
<p><img class="alignnone size-full wp-image-2854" src="{{ site.baseurl }}/assets/2019/12/CC2.png" alt="" width="853" height="482" /></p>
<ul>
<li>Next we can modify title/subtitles/descriptions/product names/URL’s etc. accordingly as per our app requirements.</li>
</ul>
<p><img class="alignnone size-full wp-image-2853" src="{{ site.baseurl }}/assets/2019/12/CC3.png" alt="" width="856" height="480" /></p>
<ul>
<li>Also we can define styles to look our products mart app more organized and eye-catching.</li>
</ul>
<p><img class="alignnone size-full wp-image-2852" src="{{ site.baseurl }}/assets/2019/12/CC4.png" alt="" width="853" height="483" /></p>
<p>Now we can see layout wise our app is bit improved and we are able to list out our products with proper icons.</p>
